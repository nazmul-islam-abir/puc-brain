# EduVault - Course & File Manager

![EduVault Banner](https://user-images.githubusercontent.com/12345678/123456789-abcdef.png) 

**EduVault** is a sleek and modern Flutter application designed to help students and educators organize their course materials efficiently. With a beautiful, dark-themed interface, it provides a centralized place to manage courses, upload files, and access your materials from anywhere.

## ‚ú® Features

- **Course Management**: Create, edit, and delete courses with custom names, descriptions, icons, and colors.
- **Semester Filtering**: Organize your view by semester with an easy-to-use dropdown filter.
- **Category Tags**: Assign categories to courses (e.g., Code, Math, Science) and filter by them.
- **File Upload & Storage**: Upload course-related files directly to a secure Supabase backend.
- **File Download & Access**: Download stored files to your device or open them directly from the app.
- **Modern UI/UX**:
    - A stunning dark theme with a vibrant, premium color palette.
    - Smooth animations and a responsive layout that works on both mobile and web.
    - A clear and intuitive navigation system with a hamburger menu and bottom navigation bar.
    - Consistent and beautiful iconography to enhance user experience.
- **Top-Tier Notifications**: Non-intrusive `MaterialBanner` notifications for success messages and validation warnings at the top of the screen.
- **Robust Validation**: All forms include input validation to prevent incorrect data entry.
